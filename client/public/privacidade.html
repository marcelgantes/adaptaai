1. Quem somos

O Adapta Aí é uma plataforma de adaptação pedagógica com inteligência artificial, desenvolvida e operada por Marcel Gantes, professor licenciado em Biologia, com sede no Rio Grande do Sul, Brasil.

Para fins da Lei Geral de Proteção de Dados (Lei 13.709/2018 — LGPD), Marcel Gantes é o Controlador de Dados desta plataforma.

Contato do DPO (Encarregado de Dados): marcelgantes@gmail.com

2. Dados que coletamos

2.1 Dados do professor (usuário)

  • Nome e e-mail — fornecidos no cadastro, via e-mail/senha ou login com Google
  • Dados de uso — turmas criadas, alunos cadastrados, adaptações geradas e histórico de atividades
  • Dados técnicos — endereço IP, tipo de navegador e dispositivo (coletados automaticamente pela infraestrutura)

2.2 Dados dos alunos

🔒 Importante: O Adapta Aí foi projetado para que nenhum dado pessoal identificável de alunos seja inserido na plataforma. O sistema utiliza apenas identificadores anônimos (apelidos ou códigos como "Aluno A" ou "Código 04").

Os dados pedagógicos dos alunos coletados são:

  • Identificador anônimo (apelido ou código)
  • Perfil pedagógico: nível de leitura, abstração, mediação, tipografia
  • Condições educacionais: dislexia, TDAH, TEA, baixa visão, altas habilidades
  • Interesses e observações pedagógicas (inseridas pelo professor)

Esses dados são dados pedagógicos de suporte ao ensino, não dados pessoais identificáveis. Ainda assim, são tratados com o mesmo cuidado exigido pela LGPD para dados sensíveis.

3. Como usamos os dados

Os dados coletados são utilizados exclusivamente para:

  • Autenticação e acesso seguro à plataforma
  • Geração de materiais pedagógicos adaptados via IA
  • Geração do Plano Educacional Individualizado (PEI)
  • Armazenamento do histórico de atividades do professor
  • Melhoria contínua da plataforma (de forma agregada e anonimizada)

Base legal (LGPD, Art. 7º): O tratamento de dados é realizado com base no legítimo interesse para prestação do serviço educacional e no consentimento fornecido no momento do cadastro.

4. Proteção de dados de alunos (LGPD)

⚠️ Responsabilidade do professor: Ao utilizar o Adapta Aí, o professor assume a responsabilidade de não inserir dados pessoais identificáveis de alunos (nome completo, CPF, endereço, foto, etc.). O sistema foi projetado para funcionar exclusivamente com identificadores anônimos.

Caso o professor insira inadvertidamente dados pessoais identificáveis, deve entrar em contato imediatamente pelo e-mail marcelgantes@gmail.com para solicitar a exclusão.

Os dados pedagógicos dos alunos são classificados como dados sensíveis pela LGPD (por incluírem informações sobre condições de saúde e desenvolvimento). Por isso:

  • São armazenados em ambiente seguro (Supabase, com criptografia em repouso e em trânsito)
  • São acessíveis apenas pelo professor que os cadastrou
  • Não são compartilhados com terceiros em nenhuma hipótese
  • Não são utilizados para treinar modelos de IA

5. Retenção e exclusão de dados

  • Os dados permanecem armazenados enquanto a conta do professor estiver ativa
  • Após a exclusão da conta, os dados são mantidos por 1 (um) ano como medida de segurança e possibilidade de recuperação
  • Após esse período, os dados são excluídos permanentemente dos nossos servidores
  • O professor pode solicitar a exclusão imediata de todos os seus dados a qualquer momento pelo e-mail marcelgantes@gmail.com

6. Compartilhamento de dados

O Adapta Aí não vende, não aluga e não compartilha dados pessoais com terceiros para fins comerciais.

Os dados são processados pelos seguintes fornecedores de infraestrutura, estritamente necessários para o funcionamento da plataforma:

  • Supabase (supabase.com) — banco de dados e autenticação, servidores nos EUA
  • Vercel (vercel.com) — hospedagem da plataforma
  • Groq (groq.com) — processamento de IA para geração de adaptações

Todos esses fornecedores possuem políticas de privacidade e segurança compatíveis com as melhores práticas internacionais. Os dados enviados para a IA (Groq) incluem apenas o conteúdo pedagógico e o perfil anônimo do aluno — nunca dados pessoais identificáveis.

7. Segurança

Adotamos as seguintes medidas de segurança:

  • Comunicação criptografada via HTTPS/TLS em todos os acessos
  • Dados em repouso criptografados no banco de dados (Supabase)
  • Autenticação segura com suporte a OAuth 2.0 (Google)
  • Políticas de Row Level Security (RLS) — cada professor acessa apenas seus próprios dados
  • Chaves de API armazenadas como variáveis de ambiente seguras, nunca expostas no código

8. Seus direitos (LGPD, Art. 18)

Como titular dos dados, você tem direito a:

  • Confirmação — saber se tratamos seus dados
  • Acesso — obter uma cópia dos seus dados
  • Correção — corrigir dados incompletos ou desatualizados
  • Anonimização ou exclusão — solicitar que seus dados sejam anonimizados ou excluídos
  • Portabilidade — receber seus dados em formato estruturado
  • Revogação do consentimento — retirar o consentimento a qualquer momento
  • Oposição — se opor ao tratamento em casos de descumprimento

Para exercer qualquer um desses direitos, entre em contato pelo e-mail marcelgantes@gmail.com. Responderemos em até 15 dias úteis.

9. Termos de Uso

9.1 Aceitação

Ao criar uma conta no Adapta Aí, você declara ter lido, compreendido e concordado com estes Termos de Uso e com a Política de Privacidade acima.

9.2 Elegibilidade

A plataforma é destinada exclusivamente a professores e profissionais da educação maiores de 18 anos. O uso por menores de 18 anos requer autorização expressa do responsável legal.

9.3 Uso permitido

  • Adaptar materiais pedagógicos para uso em sala de aula
  • Gerar PEIs para alunos sob sua responsabilidade direta
  • Compartilhar os materiais adaptados com alunos, famílias e equipe pedagógica
  • Usar os documentos gerados em contexto escolar e educacional

9.4 Uso proibido

  • Inserir dados pessoais identificáveis de alunos menores de idade
  • Usar a plataforma para fins comerciais de revenda dos materiais gerados
  • Compartilhar credenciais de acesso com terceiros
  • Tentar burlar, hackear ou sobrecarregar os sistemas da plataforma
  • Usar a plataforma para gerar conteúdo ilegal, discriminatório ou prejudicial

9.5 Propriedade intelectual

Os materiais adaptados gerados pela plataforma a partir de conteúdos enviados pelo professor são de propriedade do professor. O código-fonte, marca, logo e a plataforma em si são de propriedade de Marcel Gantes.

9.6 Disponibilidade

O Adapta Aí é fornecido "como está". Não garantimos disponibilidade ininterrupta, mas nos comprometemos a manter o serviço operacional e comunicar manutenções programadas com antecedência.

9.7 Alterações nos termos

Podemos atualizar estes termos a qualquer momento. Alterações significativas serão comunicadas por e-mail com pelo menos 15 dias de antecedência. O uso continuado da plataforma após as alterações implica aceitação dos novos termos.

10. Limitação de responsabilidade

O Adapta Aí é uma ferramenta de apoio pedagógico. O professor é o único responsável por:

  • Revisar e validar os materiais gerados pela IA antes de usá-los com alunos
  • Garantir que os dados inseridos na plataforma não incluam informações pessoais identificáveis de alunos
  • Cumprir as obrigações legais relativas ao PEI e à inclusão escolar junto à sua instituição
  • Obter as autorizações necessárias junto à gestão escolar e às famílias

A IA pode cometer erros. Os materiais gerados devem sempre ser revisados pelo professor antes do uso em sala de aula.
